### Event Streams

What are streams?

* Sequence of events happening over time

Examples of streams

* http request
* database access
* socket connection
* sensor data
* UI events: button clicks, text input
* Twitter feed
* car dashboard (speed, rpm)

Why streams?

* useful abstraction for certain types of problems
